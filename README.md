Visit: [ovatesecurity.com](https://ovatesecurity.com)

## 📋 Overview

OVATE Security's website is a high-performance, single-page application (SPA) built with semantic HTML5, Tailwind CSS (CDN), and vanilla JavaScript. It features:

- **Preloader animation** with progress bar
- **Particle canvas** background
- **Scroll-triggered reveal animations** for every section
- **Cursor spotlight** effect on service cards
- **Smooth navigation** with active-section highlighting
- **Mobile-responsive** hamburger menu
- **Contact form** with validation

## 🚀 Quick Start

The website is a **static HTML file** with no build step required.

### Option 1 — Open directly

```bash
open index.html
```

### Option 2 — Local development server

Using Python:

```bash
python3 -m http.server 8080
# Visit http://localhost:8080
```

Using Node.js (`npx serve`):

```bash
npx serve .
# Visit http://localhost:3000
```

Using VS Code: install the **Live Server** extension and click **Go Live**.
