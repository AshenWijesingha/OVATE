# Logo Assets — OVATE Security

This directory contains the official OVATE Security brand logo assets.

## Available Logos

All logos are hosted on Cloudinary at 5× resolution in `.webp` format for optimal web performance.

| File | Variant | Best Used On |
|---|---|---|
| `gray_logo_5x.webp` | Gray wordmark | Light backgrounds |
| `white_logo_5x.webp` | White wordmark | Dark backgrounds (primary) |
| `gray_logo_tm_5x.webp` | Gray + TM symbol | Official docs, light bg |
| `white_logo_tm_5x.webp` | White + TM symbol | Official docs, dark bg |
| `gray_logo_r_5x.webp` | Gray + ® symbol | Registered trademark use, light bg |
| `white_logo_tm_alt_5x.webp` | White + TM (alternate) | Dark bg, alternative layout |

## Usage Guidelines

- See [BRAND.md](../../BRAND.md) for full logo usage rules
- Minimum digital display size: **120px wide**
- Always preserve the aspect ratio — never stretch or skew the logo
- Maintain clear space equal to the height of the "O" letterform on all sides
- Do not alter colours, rotate, or add effects to the logo
