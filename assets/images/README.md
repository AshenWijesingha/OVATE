# Image Assets — OVATE Security

This directory is reserved for website imagery used in the OVATE Security website.

## Guidelines

- **Format**: Use `.webp` as the primary format for optimal compression and quality
- **Fallback**: Provide `.jpg` fallback for browsers that do not support WebP
- **Dimensions**: Export at 2× resolution for retina/HiDPI displays
- **Alt text**: Every `<img>` tag must include a descriptive `alt` attribute
- **File naming**: Use lowercase, hyphen-separated names (e.g., `hero-background.webp`)

## Recommended Tools

- [Squoosh](https://squoosh.app) — Browser-based image compression and format conversion
- [ImageOptim](https://imageoptim.com) — macOS image optimisation
- [Sharp](https://sharp.pixelplumbing.com) — Node.js image processing pipeline
